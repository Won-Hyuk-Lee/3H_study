# 🎮 ConsoleQuest: C# 기초 실습 프로젝트

이 프로젝트는 C# 프로그래밍의 기초(변수, 조건문, 연산자)를 배우기 위해 만들어진 텍스트 RPG 게임입니다.
현재 게임은 **미완성** 상태입니다. 여러분이 직접 코드를 수정해서 게임을 완성시켜 주세요!

## 🚀 실행 방법
1. **Visual Studio**나 **VS Code**에서 이 폴더(`ConsoleQuest`)를 엽니다.
2. `Program.cs` 파일을 엽니다.
3. 실행 버튼(F5)을 눌러 게임을 실행해 봅니다.
   - *현재는 좌우로만 이동할 수 있고, 상점에서는 물건을 살 수 없습니다.*

---

## 📝 과제 목록

### 과제 1: 위/아래 이동 구현하기
현재 플레이어는 **왼쪽(←)**과 **오른쪽(→)**으로만 이동할 수 있습니다.
**위쪽(↑)**과 **아래쪽(↓)** 화살표를 눌렀을 때도 이동할 수 있도록 만들어주세요.

*   **수정할 위치:** `Program.cs` 파일의 `ProcessInput` 함수 안쪽
*   **힌트:**
    *   컴퓨터 화면의 좌표는 **왼쪽 위가 (0, 0)** 입니다.
    *   오른쪽으로 갈수록 X가 커지고, **아래쪽으로 갈수록 Y가 커집니다.**
    *   `ConsoleKey.LeftArrow` 코드를 복사해서 `ConsoleKey.UpArrow`와 `ConsoleKey.DownArrow`를 만들어보세요.
    *   `playerXPosition` 대신 `playerYPosition`을 변경해야 합니다.

### 과제 2: 상점 구매 로직 완성하기
상점(집 모양 아이콘)에 들어가면 물건 목록이 나오지만, 번호를 눌러도 "기능이 완성되지 않았습니다"라고 뜹니다.
돈을 내고 물건을 살 수 있도록 만들어주세요.

*   **수정할 위치:** `Program.cs` 파일의 `EnterShop` 함수 안쪽
*   **구현해야 할 것:**
    1. **돈 확인:** 내가 가진 돈(`currentGold`)이 물건 가격(`price`)보다 많은지 확인 (`if`문 사용)
    2. **구매 처리:**
        *   내 돈에서 가격만큼 뺍니다. (`currentGold -= price`)
        *   물건의 효과를 적용합니다. (예: `currentHealth += 20`)
        *   "구매 성공!" 메시지를 띄워줍니다.
    3. **돈 부족 처리:** 돈이 부족하면 "돈이 부족합니다"라고 알려줍니다. (`else`문 사용)
