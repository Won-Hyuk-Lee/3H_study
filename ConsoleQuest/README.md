## 🚀 실행 방법
1. **Visual Studio**나 **VS Code**에서 이 폴더(`ConsoleQuest`)를 열어
2. visual studio에서는 바로 실행이 될거고 vs code에서는 확장(extension)에서 C#, C# dev kit과 닷넷6.0(https://dotnet.microsoft.com/ko-kr/download/dotnet/thank-you/sdk-6.0.428-windows-x64-installer)을 설치해야 아마 실행이 될 거임.
3. `Program.cs` 파일을 열어
4. 실행 버튼(F5)을 눌러 게임을 실행해
   - *현재는 좌우로만 이동할 수 있고, 상점에서는 물건을 살 수 없다*

---

## 📝 과제 목록

### 과제 1: 위/아래 이동 구현하기
현재 플레이어는 **왼쪽(←)**과 **오른쪽(→)**으로만 이동할 수 있어
**위쪽(↑)**과 **아래쪽(↓)** 화살표를 눌렀을 때도 이동할 수 있도록 만들어

*   **수정할 위치:** `Program.cs` 파일의 `ProcessInput` 함수 안쪽


### 과제 2: 상점 구매 로직 완성하기
상점(집 모양 아이콘)에 들어가면 물건 목록이 나오지만, 번호를 눌러도 "기능이 완성되지 않았습니다"라고 뜸
돈을 내고 물건을 살 수 있도록 만들어

*   **수정할 위치:** `Program.cs` 파일의 `EnterShop` 함수 안쪽


### 과제 3: 다양한 몬스터 만들기
현재는 **슬라임**만 나오는데 **고블린**과 **오크**도 나오게 만들어

*   **수정할 위치:** `Program.cs` 파일의 `StartBattle` 함수 안쪽


### 과제 4: 난이도 선택 기능 만들기
게임이 시작될 때 난이도를 고르고, 난이도에 따라 맵이 다르게 만들어지게 해

*   **수정할 위치:** `Program.cs` 파일의 `SelectDifficulty` 함수와 `InitializeGame` 함수
